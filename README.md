# Quiz Application (C++)

A console-based Quiz Application developed in C++ that allows users to answer multiple-choice questions, track scores, and complete quizzes within a specified time limit.

## Features

- Multiple-choice questions
- Score calculation and display
- Question timer support
- Instant answer validation
- User-friendly console interface
- Result summary at the end of the quiz
- Easy-to-add question database
- Modular code structure

## Technologies Used

- C++
- Standard Template Library (STL)
- File Handling (optional)
- Chrono Library (for timing features)


## How It Works

1. The application loads quiz questions.
2. Questions are displayed one by one.
3. The user selects an answer.
4. Answers are checked and scored.
5. A timer can be used to limit response time.
6. Final score and performance summary are displayed.
